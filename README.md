# 🏰 塔防游戏 (Tower Defense)

一款经典的 HTML5 塔防游戏，建造防御塔抵御敌人入侵！

![HTML5](https://img.shields.io/badge/HTML5-Canvas-orange?logo=html5)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow?logo=javascript)
![License](https://img.shields.io/badge/License-MIT-green)

## 🎮 游戏特色

- **纯前端实现**：HTML5 Canvas + JavaScript，无需安装
- **经典塔防玩法**：放置防御塔，阻止敌人到达终点
- **多种防御塔**：不同属性和升级路线
- **多种敌人**：不同血量和速度的敌人
- **波次系统**：越来越难的关卡挑战

## 🎯 操作方式

- **鼠标点击**：选择/放置防御塔
- **升级塔防**：点击已放置的塔进行升级
- **策略布局**：合理布置防御塔位置

## 🛠️ 运行方式

### 方式一：直接打开
```bash
# 双击 index.html 即可在浏览器中运行
```

### 方式二：本地服务器
```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

然后访问 `http://localhost:8000`
